# AI 智能施工日志生成器

输入施工信息，AI 自动生成规范的专业施工日志。

**在线体验：** https://ai-construction-log.vercel.app

## 技术栈

- Next.js 14（App Router）
- DeepSeek API

## 功能

- 输入项目名称、日期、施工部位、天气、施工内容
- AI 自动生成完整施工日志（含天气影响、人员机械、质量安全、明日计划）
- 一键复制日志内容

## 本地运行

```bash
npm install
npm run dev
```

需要配置 `DEEPSEEK_API_KEY` 环境变量。
